# ALX Travel App (Milestone 2), WEEK 4
ALX Travel App is a backend application built with Django and Django REST Framework (DRF) for a travel booking platform.  
This project demonstrates:

- **Database Modeling:** Defines models for Listings, Bookings, and Reviews with proper relationships and constraints.  
- **Serializers:** Converts model data into JSON for API endpoints.  
- **Data Seeding:** Uses Faker to populate the database with realistic sample data for testing and development.  

The app allows users to browse travel listings, make bookings, and leave reviews,
